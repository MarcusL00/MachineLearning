# CSVision

CSVision allows you to upload CSV files and train specific models on your own dataset!

The frontend is easy to use and allows quickly submitting your CSV file, to the server.

The code is expandable, allowing to easily add additional ML models.

---

To get started, clone the project and open your terminal and navigate to the [docker directory](https://github.com/MarcusL00/MachineLearning/tree/main/docker).
Then run ```docker-compose up```. Once the containers are running, you can open [localhost](http://localhost) in your browser.
